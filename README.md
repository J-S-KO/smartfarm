# 🌿 베란다 스마트팜 (Veranda SmartFarm)

라즈베리파이를 이용한 가정용 자동 급수 및 환경 모니터링 시스템입니다.

## 🚀 기능
- 온습도 실시간 로깅 (1분 간격)
- 솔레노이드 밸브 제어 (자동 급수)
- Git을 이용한 버전 관리

## 🛠️ 사용법
- **상태 확인:** `farm-check`
- **로그 보기:** `farm-log`
- **수동 급수:** `python3 valve_test.py`

## 📁 파일 구조
- `main.py`: 메인 구동 코드
- `logs/`: 데이터 저장 폴더 (Git 무시됨)
